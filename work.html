Work

Meet App

Objective: To build a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. The app uses the Google Calendar API to fetch upcoming events and displays them to users based on their city search.

Context: Meet App combines two major trends in web development: serverless architecture and PWAs. It loads instantly, works offline, and requires no backend maintenance. Using TDD ensures high-quality, reliable code. The app also features data visualizations like a scatterplot of events by location and a pie chart of event genres.

Hosted App: Visit the live Meet App

GitHub Repository: View the code on GitHub

Technologies Used: React.js, AWS Lambda, Serverless Framework, OAuth 2.0, Google Calendar API, Recharts (for data visualization), Jest, Puppeteer, React Testing Library, HTML5, CSS3, Git, GitHub Pages
